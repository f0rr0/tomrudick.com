api.tomrudick.com

This is an API for me. It is used to power my personal website but I will probably find other interesting ways to use it.

All data is aggregated from various external APIs and merged together to provide a single view into my digital life. Most of the data is in real-time or with very slight delays. More endpoints will be added as I can find the time. Additional feeds that I want to add are Github activity, daily step counts, sleep and weight data, time spent commuting per day, and inbox counts.

Endpoints

All endpoints will return JSON data regardless of any request headers and are completely public.
